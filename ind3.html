Para su bombeo de agua, se tiene diferentes tipos de configuraciones,
adaptandoese a su necesidad. A continuación se motrara cada uno de ellos.


Sistema conectado por interconexión.
Sistema interconectado y uso de batería.
Sistema aerogenerador, batería y bomba.
Sistema interconectado sin batéria. En dado que se quiera, se podría disponer de una batería más el sistema interconectado, esto es para cuando falte nuestro recurso, la CFE pueda brindarnos energía y así la batería pueda mantenerse cargada. Para los usuarios que no puedan disponer de un sistema eléctrico, para el cual este inter conectado, se tiene el sistema del aerogenerador que está conectado a la batería para que este pueda suministrar la energía, y de esta manera poder cargar la batería que va a alimentar nuestra bomba.
